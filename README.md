# SignalForge

SCOUT SYSTEM の AI駆動開発リポジトリです。
このリポジトリでは、AGENTS.md を正典として `.agents/` の進捗・フェーズ指示・技術パターンを管理します。

## 主要ファイル
- `AGENTS.md`: 全AIツール共通の正典ルール
- `.agents/progress.md`: セッション間の進捗引き継ぎ
- `.agents/phases/`: Phase 1〜6 の作業指示書
- `.agents/skills/`: 実装パターン集
- `docs/spec/scout_system_spec.md`: 仕様参照

## 言語方針
- チャット: 日本語
- AI関連ドキュメント: 英語（必要に応じて）
- 人間向けドキュメント: 日本語
- コード/コメント/命名: 英語
